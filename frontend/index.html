GitHub App webhook
Auto-trigger NEXUS when issues are opened on any repository.
Webhook URL
http://127.0.0.1:8000/api/v1/webhook/github
Setup steps
1
Go to GitHub → Settings → Developer settings → GitHub Apps → New GitHub App
2
Set the webhook URL above. Set content type to application/json.
3
Subscribe to Issues events only.
4
Install the app on your repository. NEXUS will auto-trigger on every new issue or when these labels are added:
nexus auto-fix nexus-fix ai-fix

What is SWE-bench?
SWE-bench is a benchmark of 2,294 real GitHub issues from popular Python repos (Django, Flask, Requests, etc.) with verified ground-truth patches. SWE-bench Lite is a curated subset of 300 instances designed for faster evaluation. Every major AI lab (OpenAI, Anthropic, Google) uses it to measure code generation capability. A resolution rate above 5% on Lite as a solo project is considered exceptional.
